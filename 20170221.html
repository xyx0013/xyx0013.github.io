CSS3选择符 元素选择符
  • 通配选择符 (*) 指：所有元素对象
    • * { sRules }
    • 例：*{margin:0;padding:0}
    • 通常不建议使用通配选择符，因为它会遍历并命中文档中所有的元素，出于性能考虑，需酌情使用
  • 类型选择符(E) 指：以文档语言对象类型作为选择符
    • E { sRules }
    • 例：div{....};p{....};
  • ID选择符(E#id) 指：以唯一标识符id属性等于myid的E对象作为选择符
    • E#myid { sRules }
    • 例：div#asd 或#asd
  • 类选择符(E.class) 指：以class属性包含myclass的E对象作为选择符
    • E.myclass { sRules }
    • 例：.a 或 .b
    • 不同于ID选择符的唯一性，类选择符可以同时定义多个,如：.a.b
关系选择符
  • 包含选择符(E F)
    • E F { sRules }
    • 选择所有被E元素包含的F元素
    • 与子选择符不同的是，包含选择符将会命中所有符合条件的后代，包括儿子，孙子，孙子的孙子...
  • 子选择符(E>F)
    • E>F { sRules }
    • 选择所有作为E元素的子元素F
    • 与包含选择符不同的是，子选择符只能命中子元素，而不能命中孙辈
  • 相邻选择符(E+F)
    • E+F { sRules }
    • 选择紧贴在E元素之后F元素
    • 与兄弟选择符不同的是，相邻选择符只会命中符合条件的相邻的兄弟元素
  • 兄弟选择符(E~F)
    • E~F { sRules }
    • 选择E元素后面的所有兄弟元素F
    • 与相邻选择符不同的是，兄弟选择符会命中所有符合条件的兄弟元素，而不强制是紧邻的元素
属性选择符
  • E[att]
    • E[att] { sRules }
    • 选择具有att属性的E元素
    • 例：a[class] 选择其中具有class属性的a元素
  • E[att="val"]
    • E[att="val"] { sRules }
    • 选择具有att属性且属性值等于val的E元素
    • 例：a[class="external"] 选择其中具有class属性且属性值为external的a元素
  • E[att~="val"]
    • E[att~="val"] { sRules }
    • 选择具有att属性且属性值为一用空格分隔的字词列表，其中一个等于val的E元素（包含只有一个值且该值等于val的情况）
    • 例：a[class~="external"]选择其中具有class属性且属性值为一用空格分隔的字词列表并且包含external的a元素
  • E[att^="val"]
    • E[att^="val"] { sRules }
    • 选择具有att属性且属性值为以val开头的字符串的E元素
    • 例：li[class^="a"] 选择其中具有class属性且属性值以a为开头的字符串的li元素
  • E[att$="val"]
    • E[att$="val"] { sRules }
    • 选择具有att属性且属性值为以val结尾的字符串的E元素
    • 例：li[class$="a"] 选择其中具有class属性且属性值以a为结尾的字符串的li元素
  • E[att*="val"]
    • E[att*="val"] { sRules }
    • 选择具有att属性且属性值为包含val的字符串的E元素
    • 例：li[class*="a"] 选择其中具有class属性且属性值包含a的字符串的li元素
  • E[att|="val"]
    • E[att|="val"] { sRules }
    • 选择具有att属性且属性值为以val开头并用连接符"-"分隔的字符串的E元素，如果属性值仅为val，也将被选择
    • 例：li[class|="test3"] 选择其中具有class属性且属性值以test3开头并用连接符"-"分隔的字符串的li元素，或属性值为test3的li元素
伪类选择符
  • E:link 指：设置超链接a在未被访问前的样式
    • E:link { sRules }
    • 例：a:link{} 设置a元素在未被访问前的样式
  • E:visited 指：设置超链接a在其链接地址已被访问过时的样式
    • E:visited { sRules }
    • 例：a:visited{} 指：设置a元素已被被访问过时的样式
  • E:hover 指：设置元素在其鼠标悬停时的样式
    • E:hover { sRules }
    • 例：a:hover{} 指：设置a元素在其鼠标悬停时的样式
  • E:active 指：设置元素在被用户激活（在鼠标点击与释放之间发生的事件）时的样式
    • E:active { sRules }
    • 例：a:active{} 指：设置a元素在被用户激活（在鼠标点击与释放之间发生的事件）时的样式
  • E:focus 指：设置对象在成为输入焦点（该对象的onfocus事件发生）时的样式
    • E:focus { sRules }
    • 例：input:focus {} 指：设置input元素在成为输入焦点（该对象的onfocus事件发生）时的样式
    • webkit内核浏览器会默认给:focus状态的元素加上outline的样式
  • E:lang(fr) 指：匹配使用特殊语言的E元素
    • E:lang(fr) { sRules }
    • 例：p:lang(en){} 指：设置匹配使用特殊语言的p元素的样式
  • E:not(s) 指：匹配不含有s选择符的元素E
    • E:not(s) { sRules }
    • 有了这个选择符，那么你将可以很好的处理类似这样的场景：假定有个列表，每个列表项都有一条底边线，但是最后一项不需要底边线
    • .demo li:not(:last-child) {border-bottom: 1px solid #ddd;}
  • E:root 指：匹配E元素在文档的根元素。在HTML中，根元素永远是HTML
    • E:root { sRules }
  • E:first-child 指：匹配父元素的第一个子元素E
    • E:first-child { sRules }
    • 要使该属性生效，E元素必须是某个元素的子元素，E的父元素最高是body，即E可以是body的子元素
    • li:first-child{} 设置第一个li元素的样式
  • E:last-child 指：匹配父元素的最后一个子元素E
    • E:last-child { sRules }
    • E必须是它的兄弟元素中的最后一个元素，换言之，E必须是父元素的最后一个子元素
    • li:last-child{} 设置最后一个li元素的样式
  • E:only-child 指：匹配父元素仅有的一个子元素E
    • E:only-child { sRules }
    • 要使该属性生效，E元素必须是某个元素的子元素，E的父元素最高是body，即E可以是body的子元素
  • E:nth-child(n) 指：匹配父元素的第n个子元素E，假设该子元素不是E，则选择符无效
    • E:nth-child(n) { sRules }
    • 该选择符允许使用一个乘法因子(n)来作为换算方式，比如我们想选中所有的偶数子元素E，那么选择符可以写成：E:nth-child(2n)
    • li:nth-child(2){} 设置第二个li元素的样式
  • E:nth-last-child(n) 指：匹配父元素的倒数第n个子元素E，假设该子元素不是E，则选择符无效
    • E:nth-last-child(n) { sRules }
    • li:nth-last-child(1){} 设置倒数第一个li元素的样式
  • E:first-of-type 指：匹配同类型中的第一个同级兄弟元素E
    • E:first-of-type { sRules }
    • 该选择符总是能命中父元素的第1个为E的子元素，不论第1个子元素是否为E
    • p:first-of-type{} 设置同类型中的第一个同级p元素
  • E:last-of-type 指：匹配同类型中的最后一个同级兄弟元素E
    • E:last-of-type { sRules }
    • p:last-of-type{} 设置同类型中的最后一个同级p元素
  • E:only-of-type 指：匹配同类型中的唯一的一个同级兄弟元素E
    • E:only-of-type { sRules }
    • p:only-of-type{} 设置同类型中的唯一一个同级p元素
  • E:nth-of-type(n) 指：匹配同类型中的第n个同级兄弟元素E
    • E:nth-of-type(n) { sRules }
    • p:nth-of-type(2){} 设置同类型中的第二个同级p元素
  • E:nth-last-of-type(n) 指：匹配同类型中的倒数第n个同级兄弟元素E
    • E:nth-last-of-type(n) { sRules }
    • p:nth-last-of-type(1){} 设置同类型中的倒数第一个同级p元素
  • E:empty 指：匹配没有任何子元素（包括text节点）的元素E
    • E:empty { sRules }
    • p:empty {} 设置没有任何子元素（包括text节点）的p元素
  • E:checked 指：匹配用户界面上处于选中状态的元素E。(用于input type为radio与checkbox时)
    • E:checked { sRules }
  • E:enabled 指：匹配用户界面上处于可用状态的元素E
    • E:enabled { sRules }
    • input[type="text"]:enabled {} 设置匹配用户界面上处于可用状态的input元素
  • E:disabled 指：匹配用户界面上处于禁用状态的元素E
    • E:disabled { sRules }
    • input[type="text"]:disabled {} 设置匹配用户界面上处于禁用状态的input元素
  • E:target 指：匹配相关URL指向的E元素
    • E:target { sRules }
    • .panel:target{} 设置匹配相关URL指向的.panel类选择器的样式
伪对象选择符
  • E:first-letter/E::first-letter 指：设置对象内的第一个字符的样式
    • E:first-letter/E::first-letter { sRules }
    • 此伪对象仅作用于块对象。内联对象要使用该伪对象，必须先将其设置为块级对象
    • 该伪类常被用来配合font-size属性和float属性制作首字下沉效果
  • E:first-line/E::first-line 指：设置对象内的第一行的样式
    • E:first-line/E::first-line { sRules }
    • p:first-line{}
  • E:before/E::before 指：设置在对象前（依据对象树的逻辑结构）发生的内容。
  • 用来和content属性一起使用，并且必须定义content属性
    • E:before/E::before { sRules }
  • E:after/E::after 指：设置在对象后（依据对象树的逻辑结构）发生的内容。
  • 用来和content属性一起使用，并且必须定义content属性
    • E:after/E::after { sRules }
  • E::placeholder 指：设置对象文字占位符的样式
    • E::placeholder { sRules }
    • ::placeholder 伪元素用于控制表单输入框占位符的外观，它允许开发者/设计师改变文字占位符的样式，默认的文字占位符为浅灰色
    • 当表单背景色为类似的颜色时它可能效果并不是很明显，那么就可以使用这个伪元素来改变文字占位符的颜色
    • 需要注意的是，除了Firefox是 ::[prefix]placeholder，其他浏览器都是使用 ::[prefix]input-placeholder
  • E::selection 指：设置对象被选择时的样式
    • E::selection { sRules }
    • 需要注意的是，::selection只能定义被选择时的background-color，color及text-shadow(IE11尚不支持定义该属性)
